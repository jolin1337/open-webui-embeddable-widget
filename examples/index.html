Chat Widget Demo

This page demonstrates the Open WebUI Chat Widget.

Important:

  • Make sure you have run npm install and npm run build in the project root to generate the ../dist/ChatWidget.js file.
  • Open this page in your browser.
  • To make the chat functional, you need to append query parameters to this page's URL in your browser's address bar. For example:
    • ?api_key=YOUR_API_KEY
    • ?api_key=YOUR_API_KEY&model=your_model_name
    • ?api_key=YOUR_API_KEY&endpoint=https://your.custom.endpoint/api/chat/completions

The widget will appear below.
